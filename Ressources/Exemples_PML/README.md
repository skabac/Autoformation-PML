# 📚 Exemples de fichiers PML

Bienvenue dans le dossier **Exemples_PML** ! Vous trouverez ici 10 fichiers d'exemple couvrant tous les concepts du langage PML.

## 📋 Liste des exemples

### 01. Variables et Types de Données
**Fichier :** `01_Variables_et_Types.pml`
- Variables numériques (entiers, décimaux)
- Chaînes de caractères
- Booléens
- Listes et matrices

### 02. Opérations Arithmétiques et Chaînes
**Fichier :** `02_Operations.pml`
- Opérations mathématiques (+, -, *, /, MOD)
- Manipulation de texte (UPCASE, DOWNCASE, LENGTH)
- Opérateurs de comparaison (==, <>, >, <, >=, <=)
- Opérateurs logiques (AND, OR, NOT)

### 03. Conditions IF/THEN/ELSE
**Fichier :** `03_Conditions.pml`
- Condition simple (IF/THEN/ELSE)
- Conditions multiples (ELSEIF)
- Opérateurs logiques (AND, OR, NOT)
- Conditions imbriquées

### 04. Boucles LOOP
**Fichier :** `04_Boucles.pml`
- Boucle simple (FROM/TO)
- Boucle avec pas personnalisé (BY)
- Boucle décroissante
- Boucle sur liste (IN)
- Contrôle de boucle (BREAK, CONTINUE)
- Boucles imbriquées

### 05. Fonctions
**Fichier :** `05_Fonctions.pml`
- Fonction simple (pas de paramètre)
- Fonction avec paramètres
- Fonction avec plusieurs paramètres
- Fonction avec logique interne (IF/ELSE)
- Fonction avec boucle
- Fonction pour calculs (factorielle, maximum, etc.)

### 06. Objets - Propriétés et Méthodes
**Fichier :** `06_Objets.pml`
- Propriétés des objets AVEVA (NAME, TYPE, DESC, X, Y, Z, BORE, MATL)
- Méthodes des objets (HIGHLIGHT(), DELETE(), EXISTS(), etc.)
- Simulation de manipulation d'objets
- Affichage d'informations d'objet

### 07. Listes et Collections
**Fichier :** `07_Listes_Collections.pml`
- Création de listes
- Longueur d'une liste (LENGTH)
- Accès aux éléments
- Itération sur liste (LOOP...IN)
- Filtrage (pairs, impairs)
- Recherche (minimum, maximum)
- Comptage avec condition
- Listes imbriquées
- Concaténation de listes

### 08. Projet Complet - Gestionnaire de Notes
**Fichier :** `08_Projet_Notes.pml`
- Combine variables, boucles, conditions et fonctions
- Calcul des moyennes des étudiants
- Détermination des grades
- Analyse des résultats
- Cas d'usage réel : gestion scolaire

### 09. Projet Complet - Gestion d'Inventaire
**Fichier :** `09_Projet_Inventaire.pml`
- Système de gestion de stock
- Calcul des valeurs de stock
- Alerte de stock faible
- Analyse d'inventaire
- Cas d'usage réel : logistique

### 10. Projet Complet - Analyse Statistique
**Fichier :** `10_Projet_Statistiques.pml`
- Calcul de statistiques (moyenne, min, max)
- Analyse de tendances
- Comparaison de données
- Rapport d'analyse
- Cas d'usage réel : analyse de données

---

## 🚀 Comment utiliser ces exemples

1. **Ouvrir un fichier** dans votre éditeur PML préféré
2. **Lire les commentaires** pour comprendre chaque section
3. **Exécuter le script** dans AVEVA Plant ou dans un interpréteur PML
4. **Modifier et expérimenter** pour mieux comprendre

## 💡 Recommandations d'apprentissage

**Ordre suggéré :**
1. Commencer par `01_Variables_et_Types.pml` (concepts de base)
2. Continuer avec `02_Operations.pml` (opérateurs)
3. Puis `03_Conditions.pml` et `04_Boucles.pml` (contrôle)
4. Approfondir avec `05_Fonctions.pml` (réutilisabilité)
5. Découvrir `06_Objets.pml` et `07_Listes_Collections.pml` (collections)
6. Terminer avec `08`, `09`, `10` (projets complets)

## 📝 Notes importantes

- Les exemples utilisent une syntaxe PML standard
- Les scripts sont commentés en français pour une meilleure compréhension
- Chaque exemple peut être exécuté indépendamment
- Les affichages (PRINT) permettent de suivre l'exécution
