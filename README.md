# 📚 Autoformation PML (Plant Markup Language)

Bienvenue dans ce cours d'autoformation sur le PML d'AVEVA !

## 🎯 Objectifs d'apprentissage

À la fin de ce cours, tu sauras :
- ✅ Comprendre la syntaxe fondamentale du PML
- ✅ Écrire des variables et des fonctions
- ✅ Utiliser les structures de contrôle (IF, LOOP)
- ✅ Manipuler des objets PML
- ✅ Créer des scripts d'automatisation pratiques

## 📂 Structure du cours

### 📖 Modules progressifs

| Module | Description | Durée estimée |
|--------|-------------|---|
| **01_Fondamentaux** | Syntaxe, variables, types de données | 2-3h |
| **02_Controle** | IF/THEN/ELSE, LOOP, fonctions | 2-3h |
| **03_Objets** | Manipulation d'objets, propriétés, méthodes | 2-3h |
| **04_Projets_Pratiques** | Mini-projets et cas d'usage réels | 3-4h |
| **Ressources** | Documentation, références, liens utiles | - |

## 🚀 Comment utiliser ce cours

1. **Commencer par le module 01** - Les fondamentaux
2. **Lire chaque leçon** et tester les exemples
3. **Faire les exercices** fournis
4. **Progresser vers les modules suivants**
5. **Terminer par les projets pratiques**

## 💡 Conseils

- 📝 Prends des notes pendant ta lecture
- 💻 Teste chaque exemple dans AVEVA Plant ou E3D
- 🔍 Consulte la documentation officielle en cas de doute
- 🤔 N'hésite pas à expérimenter et explorer
