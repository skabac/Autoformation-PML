# Contributing

Contributions are welcome! Here's how you can help:

## How to Contribute

1. Fork the repository
2. Create a feature branch (`git checkout -b feature/improvement`)
3. Make your changes
4. Commit your changes (`git commit -am 'Add some improvement'`)
5. Push to the branch (`git push origin feature/improvement`)
6. Create a Pull Request

## Guidelines

- Keep examples clear and well-commented
- Test all code examples
- Follow the existing structure
- Update the README if adding new content

## Content Guidelines

- Use French for all documentation
- Include practical examples
- Add exercises when relevant
- Ensure code syntax is correct

Thank you for contributing!
